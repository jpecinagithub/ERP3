# Frontend ERP3

Aplicación React + Vite del ERP3.

## Instalación

La instalación completa del proyecto está en:

- [../docs/INSTALACION.md](../docs/INSTALACION.md)

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Variables de entorno

Archivo ejemplo:

- `frontend/.env.example`

Variable principal:

- `VITE_API_URL` (por defecto: `http://localhost:3000/api`)
